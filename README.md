HollowEngine 0.3

HollowEngine 是一个使用 C++、SDL2 和 ImGui（cimgui）开发的实验性 3D 游戏引擎。

本项目从零开始开发，没有使用 Unity、Unreal Engine 或 Godot 等现成游戏引擎，旨在深入学习计算机图形学、3D 数学、游戏引擎架构以及编辑器开发。

项目目标

HollowEngine 的目标并不是与商业引擎竞争，而是打造一个属于自己的、简单直观的游戏开发工具。

该项目同时也是以下领域的学习实践：

计算机图形学

3D 数学

游戏引擎架构
